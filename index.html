Polymorphic Network (PN) Workshop 2025

Exploring the future of programmable, flexible, and service-oriented networking

Motivation Topics Program Organizers Submission

Motivation

The Polymorphic Network (PN) architecture aims to unify control and dynamically schedule forwarding behavior and resources across the network. PN extends programmable networking with multi-dimensional programmability, supporting diverse paradigms such as IP, NDN, and MobilityFirst. It enables tailored, application-specific network modalities.

Topics of Interest

  • Network architecture and theory
  • Traffic engineering and congestion control
  • Formal verification and network security
  • Monitoring, detection, and power-efficiency
  • Queue scheduling, buffer management, and table lookup
  • Testing, implementation, and big data networking
